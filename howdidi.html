a tiny css story!

Introduction
Actually, i never used to use the source to make a page before this. i often used the drag and drop functionality or the insert > elements functionality from the 'design' view.But when i began one of our projects at helloifninity, my team-mates were in some busy work that i got the slot to do the client talk and start the base designs. I should tell, i never had a bad time. First i began by using the repeat-x for an image to build a header. gradually i found the webkit methords too simple to use. i started to switch to the code view in my ide rather than the design view i used to. This time, i really got addicted i'd say. Now i cant use the design view. Its so great to use the code interface.

Implementing the facebook profile.
The facebook page is implemented in pure html-css using my own profile pic, cover image, friends, map and subscription data. The data is not dynamic. Fonts are from the google webfonts gallery. also i used the css-tricks.com for a bit of styling.

Where is my code?
You can get my code from github.com/arunanson/ . Please feel free to watch and contriibute to the project.
